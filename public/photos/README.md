# Politician Photos

This directory holds the photos for the `CongressHero` component banner.

Please add the following images (in `.jpg` or `.png` format):

1. **`rahul-gandhi.jpg`** — Displayed on the left side.
2. **`local-politician.jpg`** — Displayed in the center (larger format). Clean background recommended.
3. **`sharmila.jpg`** — Displayed on the right side.

## Format Guidelines
- Aspect ratio: Portrait (approx 4:5 or 3:4).
- Background: Preferably a clean or removed background if possible.
- The component will automatically apply a gradient overlay at the bottom and a border frame around the image.

If any image is missing, the frontend will elegantly fall back to a colored silhouette placeholder (Saffron for Rahul Gandhi, Green for Sharmila and the local candidate).
